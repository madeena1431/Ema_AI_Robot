# Resources Folder - Emma AI Robot

This Resources folder contains the audio files used by the Emma AI Robot project. These files are played at different stages of the interaction process to provide audio feedback to the user.

Files contained in this folder:

1. listen.mp3

---

Purpose:

This sound is played when the robot begins listening for user input. It indicates that the microphone is active and ready to receive speech.

2. convert.mp3

---

Purpose:

This sound is played after speech has been successfully recognized and converted into text. It indicates that the robot is processing the user's request.

## Summary:

• listen.mp3 provides feedback that the robot is actively listening.

• convert.mp3 provides feedback that speech recognition has completed successfully.

These files improve user interaction by making the robot's operating state more understandable during conversations.
